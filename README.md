# Vulnerable-Web-Application-Attack-Simulation

Purpose-built vulnerable web application for offensive security 
research and penetration testing practice.

## Attack Classes Covered
- Command Injection
- SQL Injection (SQLi)
- Cookie Hijacking
- DOM-based XSS
- Stored XSS

## Purpose
Built as a hands-on training lab to practice real-world web 
exploitation techniques and document attack chains aligned 
with OWASP Top 10.

Used internally for red team training at The CyberSapiens 
cybersecurity club, DCE.

## Stack
PHP · Python · JavaScript · MySQL · HTML · CSS · Apache

## Usage
Clone the repo and run locally in a controlled environment.
For educational and authorized testing purposes only.
